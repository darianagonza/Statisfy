Understand distribution with checkNormality()

checkNormality() helps you evaluate whether a numeric variable follows a normal distribution by taking in a numeric variable from a dataset and using it to output a histogram, a QQ plot, and a Shapiro-Wilk test result. These are typically all of the normality tests we have to run before performing statistical analysis, so this function is designed to streamline this precursive step.

In the following example, we utilize checkNormality() to visualize air temperature follows a normal distribution.

NOTE TO GRADERS: SINCE MOST OF THESE FUNCTIONS REQUIRE THE USE OF NUMERIC DATA, IT IS POSSIBLE THAT RUNNING THEM WITH THE MOUSE DATA WILL YIELD AN ERROR MESSAGE (BUILT INTO THE FUNCTIONS).

checkNormality(data = thesis_data, temp_air, line_color = "red")
> Shapiro-Wilk Test: p-value =0.0265
> `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

With the help of the variety of graphs and results output, we can see that air temperature in fact does not follow a normal distribution.

Choose Appropriate Tests with whichTest()

whichTest() helps you select the correct statistical test to use for data analysis based on the type of data the variable is, the levels of the variable that exist, and the number of variables you wish to analyze. The function will output two of the following recommendations (one parametric test and one non-parametric test):

  • ANOVA
  • Bar/Pie chart
  • Chi-Square test
  • Independent t-test
  • Kruskal-Wallis Test
  • Linear Regression
  • Mann-Whitney Test
  • Paired t-test
  • Pearson Correlation Test
  • Shapiro-Wilk Test
  • Spearman Correlation Test
  • U-test
  • Wilcoxon Signed-Ranked Test

If we were interested in the variables temp_air and ecosystem to analyze whether a relationship exists between air temperature and type of ecosystem but were unsure of which test to use, we would use whichTest() and receive an immediate and reliable recommendation.

whichTest(data = thesis_data, var1 = "temp_air", var2 = "ecosystem")
> [1] "Use a One-Way ANOVA to determine whether there are significant differences between 3+ groups (normal distribution). Use a Kruskal-Wallis Test if data is not normally distributed."

This recommendation is correct since ecosystem type is a categorical variable with 3 levels while air temperature is a numeric variable.

In the case that the function cannot make an appropriate recommendation, the following message will be displayed:

## Error in whichTest(df, var1, var2) : This variable type is not supported. Please check your variable type for errors and try again.

Identify outliers with outliers()

outliers() is an easy way to identify outliers within a dataset based on desired thresholds. By default, the function will use 99th percentile and the 1st percentile when determining what to classify as an outlier.

# Outliers that are in the above the 99th percentile and below the 1st percentile
outliers(thesis_data$temp_air, upper = 0.95, lower = 0.05)

Summary statistics with stats()

stats() provides a detailed list of summary statistics of a variable, given that it is numeric. These summary statistics are designed to make it easier to choose between more complicated forms of statistical analysis that require knowledge of data dispersion (ex: beta regression). The function will provide the following statistics:

  • Mean
  • Variance
  • Standard Deviation
  • Minimum value
  • Maximum value
  • 25th percentile
  • Median
  • 75th percentile
  • Coefficient of Variation
